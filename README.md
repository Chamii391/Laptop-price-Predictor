# Laptop Price Predictor

A simple **Machine Learning project** that predicts laptop prices based on specifications.  
The model uses Python and scikit-learn to learn from past laptop data and estimate the price of new laptops.

---

## **Features**

- Predicts laptop prices using a regression model.  
- Handles both numerical and categorical data.  
- Easy to run in Python using Jupyter Notebook or VS Code.  

---

## **Technologies Used**

- Python  
- Pandas  
- Scikit-learn  
- Jupyter Notebook / VS Code
